# Music-Player

PreReq's
- Linux OS
- raylib 
- ffmpeg 
- Have folders of mp3's in 'Music' folder in computer to simulate playlists/albums

To Run

"make && ./run" in terminal

To remove all jpg's and covers folder

"make clean" in terminal
